acoustotreams.ScalarPlaneWaveBasisByUnitVector¶

class acoustotreams.ScalarPlaneWaveBasisByUnitVector(modes)¶

Scalar plane wave basis.

A plane wave basis is defined by a collection of wave vectors specified by the Cartesian wave vector components qx, qy, and qz normalized to \(q_x^2 + q_y^2 + q_z^2 = 1\).

Plane waves can refer to:func:~acoustotreams.spw_Psi.

Parameters:

modes (array-like) – A tuple containing a list for each of qx, qy, and qz.
